About weasel-javafx-parent

This module contains the annotations, processor and tests for the javafx part of weasel.

Project Modules

This project has declared the following modules:

Name Description
weasel-javafx-annotations This is a series of annotations for use with JavaFx, that allow one to bind a fxml to a class, and connect properties and events.
weasel-javafx-processor This is an annotation processor that implements the annotations described in the javafx-annotation module.
weasel-javafx-test This is a sample project to test the javafx processor.
